This website was created in the aftermath of Covid-19. With all the free time afforded by the stay at home order, I decided to compile my travel experiences into one place. It was also a good chance to brush up on my web development skills, so I went all in. I started working on it in late October of 2020, so if you're seeing an unpolished site, just give it some time.

All the trips I've taken are in a blog format on the blog page, but if you're interested in a specific type of trip, I have links that filter the blog posts below. Enjoy!
